# websocket-template
简介：websocket客户端和服务端的使用，支持客户端断线重连，心跳等，以及简单的页面展示聊天室的实现。

## 项目模块说明:

- websocket

  async：异步方法

  chat：聊天室

  client：客户端调用

  config：websocket配置类

  constant：枚举

  controller：接口访问入口

  init：初始化

  hander：客户端处理类

  server：服务端

  thread：自定义线程池

  

## 使用：

替换成项目需要的代码即可
